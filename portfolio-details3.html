Web-Based Grievance System with Offense Detection using Linear SVC

Technologies Used: PYTHON, CSS, HTML, JavaScript and Bootstrap

Project Overview: Created a web-based platform for managing student grievances at Batangas State University, Alangilan Campus. The system features an intuitive interface for grievance submission, an admin dashboard for managing cases, real-time notifications, and integrates Linear SVC for offense detection and classification.
